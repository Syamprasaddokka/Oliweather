## Installation

1. Get a free API key at [WeatherAPI.com](https://www.weatherapi.com)
2. Clone the repo
```sh
git clone https://github.com/jareksedy/elseweather.git
```
3. Put your API key in `Config/Config.swift`
```swift
let apiKey = ""
```
4. Build, run, enjoy!
